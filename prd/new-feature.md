# 신규 기능 기획 PRD 프롬프트

## 기본 PRD 초안 생성

우리 제품은 [제품명]이고, 타겟 사용자는 [사용자 유형]입니다.

다음 기능의 PRD 초안을 작성해줘: [기능명]
아래 항목을 포함해줘:

배경 및 목적
해결하려는 문제 (사용자 Pain Point)
목표 및 성공 지표 (KPI)
기능 상세 요구사항
